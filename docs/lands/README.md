# Vad är Lands?
Lands är en av våra delservrar och den mest omfattande av dom som vi erbjuder.
Precis som namnet säger så har det med Land att göra.

Om man ska sammanfatta och förklara Lands på ett enkelt sätt så går det ut på att köpa ett område, expandera och bygga sig en stad.
Man kan hyra ut tomter, dra in skatter och på så sätt gemensamt få en stad att växa.

Det finns en del ekonomiska funktioner man behöver tänka på och det är förutom att det kostar att utöka en stad så finns där även en daglig skatt ett land måste betala till sin server.
